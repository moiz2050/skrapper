## sKrapper

sKrapper is an opensource PHP code that provide searched data from amazon in JSON format

##### Core Class File

* Core/curl.php

##### Controller file

* data_fetch.php

##### View File

* index.php

##### How To Run

* Copy project files in root of your server
* Run on your browser

```
http://localhost:8080/path-to-project/index.php
```
