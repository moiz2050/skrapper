sKrapper

sKrapper is an opensource php code that provide searched data from amazon in JSON format

File Structure
1. Core/curl.php (core class file)
2. data_fetch.php (controller file)
3. index.php (view file)

How To Run

1. Copy project files in root of your server
2. Run on your browser

http://localhost:8080/path-to-project/index.php
